# 📊 Análisis de Clientes - ConnectaTel

## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar el comportamiento de los clientes de **ConnectaTel**, una empresa de telecomunicaciones en Latinoamérica.

A través de técnicas de análisis de datos y visualización, se busca identificar patrones de uso, calidad de datos y posibles insights que ayuden a mejorar la toma de decisiones de negocio.

---

## 🎯 Objetivos

* Explorar y comprender la estructura de los datasets.
* Detectar problemas de calidad de datos.
* Realizar limpieza y transformación de información.
* Analizar patrones de comportamiento de usuarios.
* Generar visualizaciones para apoyar la interpretación de resultados.
* Obtener conclusiones y recomendaciones basadas en datos.

---

## 🛠️ Tecnologías Utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Estructura de los Datos

El proyecto trabaja con los siguientes archivos:

| Archivo           | Descripción                       |
| ----------------- | --------------------------------- |
| `plans.csv`       | Información de planes telefónicos |
| `users_latam.csv` | Datos demográficos de usuarios    |
| `usage.csv`       | Información de consumo y uso      |

---

## 📈 Proceso de Análisis

1. Carga y exploración de datasets.
2. Validación de estructura y tipos de datos.
3. Identificación de valores nulos y duplicados.
4. Limpieza y transformación de datos.
5. Análisis exploratorio de datos (EDA).
6. Generación de visualizaciones.
7. Interpretación de resultados y conclusiones.

---

## 📊 Ejemplos de Análisis

* Distribución de usuarios por plan.
* Consumo promedio por cliente.
* Análisis de valores faltantes.
* Comparación de comportamiento entre segmentos.
